###Vim使用入门：

- 启动Vim后，进入Normal模式
- 按下i键，进入Insert模式
- 返回Normal模式，按ESC键
- x键，删除当前光标所在的一个字符（Normal模式）
- ：wq，存盘+退出（:w+文件名，存盘；:q退出）
- dd,删除当前行，删除的当前行存在剪切板中
- p，粘贴剪切板
- hjkl，移动光标或使用光标键
- ：help<command>，显示相关命令的帮助
- VIM的Normal模式下，所有的键都是功能键

-------
- P，粘贴
- yy，拷贝当前行
- U，撤销
- ：e<path/to/file>，打开一个文件
- ：w，存盘
- ：saveas<path/to/file>，另存为路径
- q！,退出不保存

--------

- .（小数点），重复上一次命令
- N<command>，重复某个命令N次
